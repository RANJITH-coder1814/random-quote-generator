# random-quote-generator
A simple Random Quote Generator built using HTML, CSS, and JavaScript. When the user clicks a button, the app displays a random inspirational quote from a list

A simple web application that generates a random inspirational quote when the user clicks a button.

## 🚀 Features

* Generate random quotes
* Simple and clean user interface
* Beginner-friendly JavaScript project

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
random-quote-generator
│
├── index.html
├── style.css
└── script.js
```

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Click **New Quote** to generate quotes
